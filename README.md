## ✨ 功能概览

### 📊 策略选股

基于多维度技术指标和基本面因子的智能选股引擎，支持多种经典策略：

- **多策略并行扫描** — 内置多种策略（趋势跟随、超跌反弹、动量突破、稳健成长等），一键扫描全市场 A 股
- **智能评分排序** — 每只股票按策略规则综合打分，自动去重，保留最高评分
- **策略速览** — Dashboard 展示各策略命中 Top 10，一目了然
- **策略详情** — 查看每只股票的评分明细（涨跌幅、成交量确认、估值因子等）
- **因子可视化** — 分数条、进度条、标签等直观展示各维度表现

### 📈 ETF 分析

- **ETF 行情扫描** — 覆盖市场主流 ETF，实时价格与涨跌幅
- **ETF 评分系统** — 基于趋势强度、成交量、波动率等多因子综合评估
- **自选 ETF 监控** — 添加关注的 ETF 到自选列表，随时跟踪

### 💰 基金分析

- **基金净值查询** — 查看基金历史净值走势
- **净值走势图** — 可视化基金净值变化曲线，支持多周期对比
- **自选基金监控** — 管理关注的基金列表，分类查看

### ⭐ 自选管理

- **自选股票** — 从策略扫描结果一键添加，或手动输入代码
- **自选 ETF** — 独立管理 ETF 关注列表
- **自选基金** — 独立管理基金关注列表
- 三大自选池独立管理，互不干扰

### 📉 K 线图表

- **日 K 线图** — 交互式股票 K 线图表，支持缩放拖拽
- **分时图** — A 股交易时段实时分时走势
- **多周期切换** — 支持不同时间维度查看

## ⚙️ 安装说明

### Windows

双击 `AI-Stock-Picker-Setup-1.0.0.exe`，按提示完成安装。支持自定义安装路径。

### macOS

1. 根据芯片类型下载对应的 `.dmg` 文件
2. 打开 `.dmg`，将「智能选股」拖动到 `Applications` 文件夹
3. 首次打开如提示「无法验证开发者」，前往 **系统设置 → 隐私与安全性** → 点击「仍要打开」

### Android

1. 下载 `.apk` 文件到手机
2. 在 **设置 → 安全** 中开启「允许安装未知来源应用」
3. 点击 APK 文件完成安装
