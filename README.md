## ⚙️ 安装说明

**Windows**
双击 `AI-Stock-Picker-Setup-1.0.0.exe`，按提示安装，支持自定义安装路径。

**macOS**
打开 `.dmg` 文件，将「智能选股」拖动到 `Applications` 文件夹即可。
> 首次打开如提示「无法验证开发者」，请前往 **系统设置 → 隐私与安全性** 点击「仍要打开」。

**Android**
下载 `.apk` 文件，在手机设置中允许「安装未知来源应用」，然后点击安装。
